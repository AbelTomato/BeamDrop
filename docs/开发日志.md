## 2026-06-29 12:21 - 协议层 MVP

### 实现内容

- 新增 `PacketType`，定义 MVP 阶段消息类型：
  - `Hello`
  - `FileInfo`
  - `ResumeReq`
  - `ResumeAck`
  - `Data`
  - `FileEnd`
  - `Finish`
  - `Error`
  - `Heartbeat`
- 新增 `PacketHeader`，定义固定 52 字节协议头：
  - `Magic`：`BDRP`
  - `Version`：`1`
  - `Type`
  - `Flags`
  - `Length`
  - `Checksum`
- 新增 `Packet`，组合 header 与 payload。
- 新增 `Serializer::encode()`，将 packet 编码为网络字节序字节流。
- 新增 `PacketParser::parse()`，从字节流解析 packet，并校验：
  - 最小长度；
  - magic；
  - version；
  - packet type；
  - payload 长度一致性。
