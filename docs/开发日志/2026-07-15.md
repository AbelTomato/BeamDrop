# 2026-07-15 开发日志

## Slice 5：REST 权威快照 API

### 实现内容

- 引入 `fastapi`、`uvicorn`、`pydantic` 与 `httpx` 开发依赖，新增 FastAPI 应用工厂 `app.main:create_app`。
- 新增 Pydantic wire DTO，显式把 snake_case REST JSON 转换为 Slice 4 内存领域 DTO，避免 HTTP schema 进入 `TaskManager` 或 native binding。
- 实现 health、权威 snapshot、异步发送任务、接收服务 start/stop 以及单任务查询 API；任务和可能阻塞的接收服务调用均不阻塞 asyncio event loop。
- 实现稳定的 JSON 4xx 错误体；自定义 Pydantic validator 的 Python 异常上下文在输出前剥离，防止 `ValueError` 造成错误响应序列化失败。
- CORS 仅放行两个 Vite 开发 origin；新增 `backend/run_windows_dev.py` 处理当前 MinGW `.pyd` 与 runtime DLL 的开发启动路径，优先从 `PATH` 发现 `g++.exe`，也可用 `BEAMDROP_MINGW_BIN` 显式指定。
